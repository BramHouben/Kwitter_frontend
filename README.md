# Kwitter Frontend

kwitter is een applicatie die een replica van het sociale netwerk Twitter is.
De applicatie is een semester 6 project gemaakt door Bram Houben van s63

## Installatie applicatie:

Installatie
`npm install`

Run de testen
`npm test`

Maak de productie byuld
`npm run build`
